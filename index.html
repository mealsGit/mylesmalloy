moodtunes

Completed in 2014, updated in 2018

Moodtunes is a web app I created as a school project that plays small bits of my own music based on criteria selected from a form submission.

I recently went back in 2018 to add some "quality of life" improvements like allowing users to select as many or as few musical attributes as desired before submitting the form, as well as updating the visual elements to be more sleek.

Built with: HTML5, CSS3, Javascript/jQuery, PHP, SQL, Illustrator, Garageband.
